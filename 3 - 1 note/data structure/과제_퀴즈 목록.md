퀴즈 1 : 주어진 알고리즘의 시간복잡도 구하기. `O(logn)`
```cpp
int algorithm(int n) {
	int k = 0;
	while (n > 1) {
		n = n / 2;
		k++;
	}
	return k;
}
```
